Understanding User Retention Analysis

User retention analysis measures how well a product or service maintains its user base over time. It helps businesses understand which factors keep users engaged, identify patterns of disengagement, and develop strategies to reduce churn and increase lifetime value.

Key Metrics

  • Retention Rate - Percentage of users who continue using the product after a specific period
  • Churn Rate - Percentage of users who stop using the product during a specific period
  • Customer Lifetime Value (CLV) - The total revenue a business can expect from a customer during their relationship
  • N-day Retention - Percentage of users who return on the Nth day after their first use
  • Stickiness - Ratio of daily active users to monthly active users (DAU/MAU)

Python Code Example


import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
import seaborn as sns

# Create the DataFrame directly
df = pd.DataFrame({
    'Month 1': [100, 100, 100, 100, 100, 100],
    'Month 2': [60, 65, 62, 68, 70, 0],
    'Month 3': [40, 42, 41, 45, 0, 0],
    'Month 4': [30, 35, 32, 0, 0, 0],
    'Month 5': [28, 30, 0, 0, 0, 0],
    'Month 6': [25, 0, 0, 0, 0, 0]
}, index=['Jan', 'Feb', 'Mar', 'Apr', 'May', 'Jun'])

# Create heatmap
plt.figure(figsize=(10, 8))
sns.heatmap(df, annot=True, cmap="YlGnBu", fmt='g')
plt.title('User Retention by Cohort (%)', fontsize=16)
plt.xlabel('Months Since Acquisition')
plt.ylabel('Acquisition Cohort')
plt.tight_layout()
plt.show()

Survival Curve Analysis (Kaplan-Meier Estimator)

Survival curves visualize the probability of users remaining active over time, accounting for users who haven't yet churned.

Example

This survival curve shows the probability of user retention over a 12-month period, with confidence intervals to indicate uncertainty.

Python Code Example


import pandas as pd
from lifelines import KaplanMeierFitter
import matplotlib.pyplot as plt

# Sample data
data = {
    'duration': [1, 3, 5, 7, 9, 11, 15, 20, 25, 30, 40, 50],
    'event_observed': [1, 1, 1, 0, 1, 0, 1, 1, 0, 1, 1, 0]  # 1 = churn, 0 = still active
}

df = pd.DataFrame(data)

# Fit the Kaplan-Meier model
kmf = KaplanMeierFitter()
kmf.fit(df['duration'], df['event_observed'], label='User Retention')

# Plot the survival curve
plt.figure(figsize=(10, 6))
kmf.plot_survival_function(ci_show=True)

plt.title('User Retention Survival Curve', fontsize=16)
plt.xlabel('Days Since Acquisition')
plt.ylabel('Survival Probability')
plt.grid(True, alpha=0.3)
plt.tight_layout()
plt.show()

Case Studies

SaaS Platform Retention Optimization

Problem: A B2B SaaS company was experiencing high churn rates in the second month after customer onboarding.

Analysis: Cohort analysis revealed that users who didn't use specific advanced features during their first 30 days were 3x more likely to churn in month 2.

Solution: The company redesigned their onboarding process to include guided feature tours and implemented an email sequence highlighting advanced features before the 30-day mark.

Result: 60-day retention improved by 25%, and annual customer lifetime value increased by 30%.
